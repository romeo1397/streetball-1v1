# 街球1v1 · StreetBall 1v1

**🎮 在线试玩 / Play now: https://romeo1397.github.io/streetball-1v1/**

免费 3D 街头篮球网页游戏——半场 1v1 单挑 AI。2K 式操作手感，主打**浮夸的晃倒（Ankle Breaker）与隔扣（Posterize）演出**。单文件、零依赖、离线可玩。

A free 3D streetball basketball web game — halfcourt 1v1 against AI, with 2K-style controls, exaggerated ankle breakers, posterizing dunks and chase-down blocks. Single HTML file, zero dependencies, works offline.

## 特性 Features

- 🏀 **半场 1v1**：11/21 分制、14 秒进攻、赢球得球权街球规则；另有**训练模式**（无限球权随便练）
- 🕺 **六种花式运球**：体前变向 / 背后 / 胯下 / 转身 / 犹豫步 / 后撤步，连招加成，高频晃倒防守人
- 💥 **12 式扣篮 + 扣篮拉条**：战斧 / 风车 / 360° / 720° / 胯下换手 / 挂臂 / 罚球线滑翔……油漆区冲刺无限起飞，绿区外扣飞
- 🖐️ **盖帽与追帽**：LeBron 式追身钉板大帽，慢动作演出
- 🎯 **背身单打**：一键背打，虚幻转身 / 后仰跳投 / 勾手 / 背转身突破
- 👥 **6 名可选球员**：闪电 / 神射 / 暴扣王 / 全能王 / 巨塔 / 街球手，数值影响速度、命中、扣篮窗口、抢断盖帽与晃倒率
- 🎚️ **标准（2K式吃干扰）/ 娱乐** 双投篮判定，4 档 AI 难度（新手 → 名人堂）
- 🔊 全程序化生成：建模、动画（45+ 动作片段）、音效（WebAudio）均为代码生成，无任何外部资源

## 操作 Controls

| 键 | 进攻 | 防守 |
|---|---|---|
| WASD | 移动 | 滑步 |
| Shift | 冲刺 | 快速滑步 |
| 空格 | 蓄力投篮（松手出手）；近筐冲刺=扣篮（空中松手定拉条） | 盖帽 |
| J / K / L / U / I / O | 花式：变向/背后/胯下/转身/犹豫步/后撤步 | J 抢断 K 盖帽 |
| E | 一键背打（背打中 J 虚幻转身 · K 后仰 · L 勾手 · U 背转突破） | — |
| F | 假投 | — |
| P / Esc | 暂停 | 暂停 |

## 技术 Tech

Three.js r128 · 单 HTML 文件（~940KB 内嵌引擎）· 程序化骨骼动画系统 · 无构建依赖，克隆后双击 `index.html` 即玩。
